Tabs

Here you may include a new parameter: 'includeResponsiveDropdown'. It will add a new functionality for responsives screens.


  
    const tab = new Collapsify({
      nameSpace: "tab",
      closeOthers: true,
      includeResponsiveDropdown: true,
  });

Control Button: data-{different}-control="{ID}" * multiple elements Collapsible element: data-{different}-content="{ID}" * only one element
